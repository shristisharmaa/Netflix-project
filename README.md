📊 Netflix Media Analytics Pipeline

This project is an end-to-end data analytics pipeline built using Python, SQL, and Streamlit.
 
It allows users to upload a Netflix dataset, process the data, store it in a database, and perform SQL-based analysis to extract meaningful insights.

🚀 Features
📂 Upload Netflix CSV dataset
🗄 Store data in SQLite database
📊 Perform SQL-based analysis
🔍 Predefined insights:
Top genres
Top-rated content
Content distribution by year
🧠 Custom SQL query engine
📈 Interactive UI using Streamlit
🛠 Tech Stack
Python
Pandas
SQLite
Streamlit
📌 Use Case

This project demonstrates how data engineers and analysts process structured datasets, perform transformations, and generate insights using SQL.

▶️ How to Run
pip install -r requirements.txt
python -m streamlit run app.py
